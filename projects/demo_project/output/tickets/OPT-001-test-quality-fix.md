### 任务 1: 修复流式响应问题

**描述**: 将 XLSX 和 PDF 的流式响应改为 buffer 方式

**修改文件**:
- `src/backend/src/controllers/importExport.js`
- `src/backend/src/controllers/print.js`

**修改示例**:
```javascript
// 修改前
XLSX.write(wb, { type: 'stream', bookType: 'xlsx' }).pipe(res);

// 修改后
const buffer = XLSX.write(wb, { type: 'buffer', bookType: 'xlsx' });
res.setHeader('Content-Type', 'application/vnd.openxmlformats-officedocument.spreadsheetml.sheet');
res.setHeader('Content-Disposition', 'attachment; filename="devices.xlsx"');
res.send(buffer);
```
