# 运维反馈：操作日志需求

**反馈时间**: 2026-03-23T00:00:00  
**优先级**: P1  
**类型**: 安全需求

---

## 运维反馈

系统需要记录操作日志，用于审计追溯。

## 需求背景

> "公司安全审计要求所有数据变更都要有日志记录，包括谁在什么时候做了什么操作。"
> —— 运维部门

## 功能需求

### 记录内容
- 操作人
- 操作时间
- 操作类型（新增/修改/删除）
- 操作对象
- 变更前后数据

### 日志查询
- 按时间范围查询
- 按操作人查询
- 按操作类型查询

### 日志保留
- 保留至少 90 天
- 支持导出

## 验收标准

- [ ] 所有 CRUD 操作都有日志
- [ ] 可以查询操作日志
- [ ] 日志包含变更前后数据
- [ ] 日志不可修改

## 优先级

**P1** - 应该实现
