# 用户反馈：批量操作需求

**反馈时间**: 2026-03-23T00:00:00  
**优先级**: P1  
**类型**: 功能需求

---

## 用户反馈

设备管理员反馈：当需要批量修改设备状态时，目前只能逐个操作，效率很低。

## 场景描述

> "我们每季度都要盘点设备，需要把一批设备标记为'闲置'或'报废'，现在只能一个个点，太慢了。"
> —— 某企业设备管理员

## 功能需求

### 批量选择
- 列表支持多选
- 全选/反选
- 显示已选数量

### 批量操作
- 批量修改状态
- 批量修改部门
- 批量删除（需二次确认）

## 验收标准

- [ ] 可以多选设备
- [ ] 可以批量修改状态
- [ ] 批量操作有确认提示
- [ ] 操作后显示成功/失败数量

## 优先级

**P1** - 应该实现
