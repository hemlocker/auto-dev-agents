# 设备管理系统

## 项目背景

需要一个简单的设备管理系统，用于管理企业设备信息。

## 核心功能

设备增删改查（CRUD）

## 设备字段（10个）

| 序号 | 字段名 | 说明 | 类型 |
|------|--------|------|------|
| 1 | 设备编号 | 唯一标识 | 文本 |
| 2 | 设备名称 | 设备名称 | 文本 |
| 3 | 设备类型 | 设备分类 | 下拉选择 |
| 4 | 规格型号 | 规格参数 | 文本 |
| 5 | 购买日期 | 采购时间 | 日期 |
| 6 | 使用部门 | 所属部门 | 文本 |
| 7 | 使用人 | 责任人 | 文本 |
| 8 | 设备状态 | 在用/闲置/报废 | 下拉选择 |
| 9 | 存放位置 | 设备位置 | 文本 |
| 10 | 备注 | 其他说明 | 多行文本 |

## 功能要求

### 基础功能
- [x] 设备列表展示（支持搜索、筛选）
- [x] 新增设备
- [x] 编辑设备
- [x] 删除设备
- [x] 设备详情查看

### 交互要求
- 列表支持分页
- 支持按设备类型、状态筛选
- 支持关键字搜索
- 删除前需确认

## 技术要求

- 界面简洁易用
- 响应式设计
- 数据验证（必填项、格式校验）
- 错误提示友好

## 设备类型（预设）

1. 服务器
2. 电脑
3. 打印机
4. 网络设备
5. 其他

## 设备状态

1. 在用
2. 闲置
3. 报废

## 优先级

**P0** - 全部功能必须实现

## 验收标准

1. 可以完成设备的完整CRUD操作
2. 列表显示正常，分页可用
3. 筛选和搜索功能正常
4. 数据验证有效
5. 界面响应式，移动端可用
