# 新功能需求：设备导入导出

## 功能描述

支持批量导入和导出设备数据，提高数据录入效率。

## 用户故事

**作为** 设备管理员  
**我想要** 批量导入和导出设备数据  
**以便于** 快速录入大量设备和备份数据

## 功能需求

### 1. Excel 导入

- 支持下载导入模板
- 支持 .xlsx 格式
- 自动验证数据格式
- 显示导入结果（成功/失败条数）
- 失败数据给出具体错误原因

### 2. Excel 导出

- 导出当前筛选结果
- 支持 .xlsx 格式
- 包含所有字段

### 3. 导入模板

- 提供标准模板下载
- 包含示例数据行
- 字段说明清晰

## 接口设计

```
POST /api/v1/devices/import    # 导入设备
GET  /api/v1/devices/export    # 导出设备
GET  /api/v1/devices/template  # 下载模板
```

## 验收标准

- [ ] 可以下载导入模板
- [ ] 可以成功导入有效数据
- [ ] 无效数据给出明确错误提示
- [ ] 可以导出设备列表
- [ ] 导出的数据可以再导入
